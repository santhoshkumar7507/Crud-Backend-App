👁️ How to View Your Results
Three ways to run Locust and see results live:

🌐 Option 1: Locust Web UI (Visual Dashboard)
python -m locust -f locustfile.py --host http://localhost:3001
Then open http://localhost:8089 in your browser → Enter users + ramp rate → Click Start
⚡ Option 2: Headless (Terminal output)
python -m locust -f locustfile.py --headless -u 10 -r 2 -t 15s --host http://localhost:3001
📊 Option 3: Export CSV Report
python -m locust -f locustfile.py --headless -u 10 -r 2 -t 15s --host http://localhost:3001 --csv=my_report --html=my_report.html
Generates my_report.html — open in browser for charts + full stats
